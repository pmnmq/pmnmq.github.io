Ubuntu环境下操作

环境：Ubuntu 20.04.1

示例：hexo博客框架生成的静态文件

hexo框架生成静态文件并上传

1

hexo g

生成的静态文件在public文件夹中

将public文件夹打包为zip格式，上传到服务器，我这里放到的是用户目录下，我是root用户

如果静态文件中有中文名字，解压时需要指定编码

1

unzip -O GBK public.zip

然后检查public文件夹下是否还是public文件夹

安装nginx

安装openssl

1
2

apt install openssl
apt install libssl-dev

1

wget http://nginx.org/download/nginx-1.18.0.tar.gz

解压文件夹，进入目录

1
2
3

./configure --prefix=/usr/local/nginx --with-http_stub_status_module --with-http_ssl_module
make && make install
ln -s /usr/local/nginx/sbin/nginx /usr/bin/nginx

启动nginx

1

nginx

配置nginx

修改配置文件

1

nvim /usr/local/nginx/conf/nginx.conf

将第一行用户修改为当前用户

1
2

-user nobody;
+user root;

修改location中托管文件地址

1
2

-root /root/html;
+root /root/public;

保存退出后nginx重新加载配置文件

1

nginx -s reload

然后访问公网ip就能看到页面了
