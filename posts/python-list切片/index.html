Python中的list对象是Python中最常用的数据类型，而list切片是让list对象变得灵活且方便操作的对象，对！切片也是一种对象！

切片是我们中文的称呼，官方的叫法是slice！

slice的运用

要想明白slice的实现原理，肯定要先明白slice的运用。 切片模式：[start:end:step]

  • start表示切片开始的位置，默认为0
  • end表示切片截至（但不包含）位置（默认为列表长度）
  • step表示切取数据的步长（默认为1）
  • 当start为0时可以省略，当end为列表长度时可以省略
  • 当step为1时可以省略，并且省略步长时可以同时省掉一个冒号
  • 当step为负整数时，表示反向切片，这时start应该要比end的值大才行

1
2
3
4

>>> int_list = [3, 4, 5, 6, 7, 8, 9, 11, 13, 15, 17]
>>> int_list[1:2] = ['a', 'b']
>>> int_list
[3, 'a', 'b', 5, 6, 7, 8, 9, 11, 13, 15, 17]

其实上面的三种插入元素的方式叫做设置更合适，总结为以下两点：

  • 当冒号前后位置相同时，在原列表指定位置前插入指定元素
  • 当位置不同时，将指定元素设置到原列表指定位置

这里就报错了，原因是左边的切片长度是5，而右边的切片长度是3。

在进行切片操作的时候（[start:end:step]），Python会自动将切片的参数初始化成一个slice对象，然后将这个slice对象传递给item，但是传递的是索引（[0]），那么这个对象就是一个int类型，这样就不是切片。要判断一个对象的类型，可以使用isinstance来判断

        self.college_name = college_name
        self.class_name = class_name
        self.students = students
        
    def __getitem__(self, item):
        cls = type(self)
        # cls = College
        if isinstance(item, slice):
            return cls(college_name = self.college_name, class_name = self.class_name, students = self.students[item])
        elif isinstance(item, numbers.Integral):
            # 注意College类接收的是带[]的，索引得到的值要加上[]
            return cls(college_name = self.college_name, class_name = self.class_name, students = [self.students[item]])

class College:
    
    def __init__(self, college_name, class_name, students):
        self.college_name = college_name
        self.class_name = class_name
        self.students = students
        
    def __getitem__(self, item):
        cls = type(self)
        # cls = College
        if isinstance(item, slice):
            return cls(college_name = self.college_name, class_name = self.class_name, students = self.students[item])
        elif isinstance(item, numbers.Integral):
            # 注意College类接收的是带[]的，索引得到的值要加上[]
            return cls(college_name = self.college_name, class_name = self.class_name, students = [self.students[item]])
        
        
    def __repr__(self):
        return repr(self.students)
    
    def __reversed__(self):
        return students.reverse()
    
    def __len__(self):
        return len(self.students)
    
    def __iter__(self):
        return iter(self.students)
    
    def __contains__(self, item):
        if item in self.students:
            return True
        else:
            return False
        
    def append(self, p_object):
        self.students.append(p_object)
    
    def extend(self, iterable):
        self.students.extend(iterable)


students = ['李华', "张三", "王五", "王大锤"]

结语

想让上面实现的可切片的类看起来更像list，可以参考list的源码，这里的重点是讲解如何实现一个可切片的对象

在Python中实例化一个对象的时候是这样的obj = Class()

而上面的程序中想让College对象都是College类型的，使用的是类型转换去初始化对象，而我们在平时的程序中，要将一个其他类型的变量转换成str类型的变量使用的肯定是obj = str(obj)，这很明显就是一个str对象初始化的过程，lis = list()初始化一个空列表也是一样的。

在Python中一切皆对象，使用一切皆对象的思维来看Python程序，那么程序的设计过程将变得简单，也更容易写出Python风格的代码。
